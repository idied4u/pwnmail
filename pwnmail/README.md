# pwnmail

A simple, private, and free encrypted email tool for real people. No surveillance. No lock-in. No paywalls.

# Requirements
python-gnupg
qrcode
rich
python-dotenv
Pillow
stepic

# Usage demo
Example:
$ sendsecure init
$ sendsecure send --to someone@example.com --subject 'Encrypted' --body 'Hello world'
